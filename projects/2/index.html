< Workshop Client />

Description

Prototype d'une Application Web conçu pour le ministère de l'éducation nationale. Le but de ce workshop était de concevoir et développer en une semaine un outil mettant en relation les particuliers en recherche d'apprentis et les lycées professionnels. Sur cette application, les particuliers peuvent trouver des informations sur tous les lycées professionnels de France, ainsi que les formations qu'ils proposent.
